# Workflow: Handoff

Use this workflow when another teammate or AI tool needs to continue the work.

## Handoff Template

```markdown
## Objective

<What we are trying to accomplish>

## Current State

<What exists now>

## Relevant Docs

- <doc path>

## Files Changed

- <file path>: <why>

## Verification

- <command>: <result>

## Open Questions

- <question or none>

## Next Steps

1. <next concrete action>
```

Keep handoffs short enough to paste into any AI tool.
