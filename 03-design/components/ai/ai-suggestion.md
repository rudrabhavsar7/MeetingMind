# MeetingMind Component: AI Suggestion (Quick Prompt)

## 1. Overview
Clickable pre-written prompts offered to the user to demonstrate the capabilities of the AI Search.

## 2. Design Philosophy
Blank search boxes suffer from the "Blank Page Problem." Users often don't know *what* they can ask an AI. Suggestions bridge that gap.

## 3. Problem Statement
A user opens the "Ask AI" tab and types "Hello?", wasting an interaction.

## 4. UX Goals
* Educate the user on system capabilities.
* Provide one-click value.

## 5. Usage Guidelines
* Displayed below the `AISearchInput` when the input is empty.

## 6. When to Use
* Empty state of RAG Search.
* Meeting details page ("Suggested Questions").

## 7. When NOT to Use
* While the user is already typing a custom query.

## 8. Component Anatomy
* Container (Flex wrap).
* Suggestion Chip (Rounded pill, slightly larger than a standard Chip).

## 9. Variants
* Outline (Default).
* Subtle background.

## 10. Sizes
* `text-sm`, comfortable touch target.

## 11. States
* Hover (Background darkens, cursor pointer).

## 12. Layout Rules
* `flex flex-wrap gap-2`.

## 13. Content Guidelines
* Phrased as a complete question from the user's perspective (e.g., "What were the key takeaways?").

## 14. Icon Rules
* Can include a leading icon relevant to the question (e.g., a Checkmark for action items, a Clock for timelines).

## 15. Color System
* `border-border text-foreground hover:bg-muted`.

## 16. Typography
* `text-sm`.

## 17. Spacing
* `px-4 py-2 rounded-full`.

## 18. Motion
* Subtle scale-up or border color change on hover.

## 19. Accessibility
* Render as `<button>` elements.

## 20. Keyboard Interaction
* Tab and Enter.

## 21. Responsive Behavior
* Wraps naturally.

## 22. Dark Mode
* Inherited.

## 23. Design Tokens
* `--border`, `--muted`.

## 24. API Specification
```tsx
<div className="flex flex-wrap gap-2 mt-4">
  <button 
    onClick={() => setPrompt("What are the action items?")}
    className="rounded-full border px-4 py-2 text-sm transition-colors hover:bg-muted"
  >
    What are the action items?
  </button>
</div>
```
